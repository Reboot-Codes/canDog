## Candog

Candog for a Flipper Zero application, you can use this code as a starting point for developing your own app. 
### Features
- Start Screen
- Menu
- Button Menu
- File Browser
- Different Scenes / Views
- Settings Page (On/Off for haptics, sound, led)
- Handling of Button Inputs
- Handling of Tick events
- Handling of Enter/Exit events
- Rendering to GUI
- Haptic Feedback 
- Sound Effects
- LED Effects
- Dolphin Deeds (make dolphin happy)
- Storage (Saves settings)
